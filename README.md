# Installing
```
git clone https://github.com/Articuno1234/plusbot
```

# Setting up
Open up the Plus Folder then go into `app`
you will see a file called `config.py` open it and put all your bots details into it

# Running
Normal
```
cd plusbot
python3 run.py
```
BETA
```
cd plusbot
```
